Zufallszahlen und zufällige Elemente mit dem Random-Modul in Python

Mithilfe des Random-Moduls in Python können wir Zufallszahlen generieren und zufällige Elemente aus einer Liste auswählen.

Um die Methoden zu nutzen, müssen wir zuerst das Modul importieren, indem wir am Anfang unseres Codes den Befehl import random schreiben.

random.randint(a, b)

Diese Funktion gibt eine zufällige ganze Zahl (Integer) zwischen den Werten a und b zurück. Beide Werte sind einschließlich (d.h. a und b können ebenfalls ausgewählt werden).

Beispiel:

import random

a = random.randint(50, 70)
print(a)  # Ausgabe: Eine zufällige Zahl zwischen 50 und 70 (z. B. 63)

random.choice(list)

Diese Funktion wählt ein zufälliges Element aus einer gegebenen Liste aus und gibt es zurück.

Beispiel:

import random

warenkorb = ["Apfel", "Fleisch", "Tee"]
w = random.choice(warenkorb)
print(w) # Ausgabe: Ein zufälliges Element aus der Liste (z. B. "Fleisch")

Wahrheitswerte: True und False

True und False sind boolesche Werte, die in Python verwendet werden, um Wahrheitswerte darzustellen.

Beispiel:

if True:
    print("Hallo") # Ausgabe: Hallo

if False:
    print("Hallo") # Keine Ausgabe
  • if True: führt den Code innerhalb der Bedingung aus, weil die Bedingung trivialerweise wahr ist.
  • if False: überspringt den Code, weil die Bedingung trivialerweise falsch ist.
