Lists in Python

A list is a collection of items that you can store in a variable.


A list can contain different data types, such as numbers, strings, or even other lists.


Lists are useful because they allow you to group multiple values in one structure and access them easily.

  • Lists are created with square brackets [] and individual elements are separated by commas.
  • You can access elements using an index. The indices start at 0, meaning the first element has index 0, the second 1, and so on.

Example 1:

fruits = ["Apple", "Banana", "Cherry"]
print(fruits)
print(fruits[0])  
print(fruits[1])  
print(fruits[2])


Output 1:

['Apple', 'Banana', 'Cherry']
Apple
Banana
Cherry


You can add or remove items in a list to adjust its content.

  • Use the append() method to add a new item at the end of the list.
  • To remove a specific item, you can use the remove() method.

Example 2:

shopping_list = ["Bread", "Milk", "Eggs"]
print(shopping_list) 
shopping_list.append("Butter") 
print(shopping_list) 
shopping_list.remove("Milk")
print(shopping_list)


Output 2:

['Bread', 'Milk', 'Eggs']
['Bread', 'Milk', 'Eggs', 'Butter']
['Bread', 'Eggs', 'Butter']


You can change items in a list, for example by doubling an element, or swapping items in different positions.


Example 3:

data = [5, "Red", 12, "Blue"]
# Double the third element
data[2] = data[2] * 2 
print(data)

# Swap the first and last elements
temp = data[0]
data[0] = data[3]
data[3] = temp 
print(data)


Output 3:

[5, 'Red', 24, 'Blue']
['Blue', 'Red', 24, 5]


Explanation: To swap two elements, we first store the first element in an extra variable temp. Then we set the last element in the place of the first, and finally, place the saved element in the last position. If you simply try the following, the code will not work as expected:

data[0] = data[3]
data[3] = data[0]

In this case, the value at the first position will be directly overwritten before it can be placed at the last position. Therefore, an additional variable is necessary to temporarily store the original value.

The for Loop

A for loop is used to go through the elements of a list or string one by one and perform an action for each element.


The loop is repeated until all elements of the list or characters of the string have been processed.


Example 1:
In this code, a list of fruits is created, and the for loop prints each fruit individually.

fruits = ["Apple", "Banana", "Cherry"]
for x in fruits:
  print(x)


Output 1:

Apple
Banana
Cherry


Example 2:
In this code, each name in the list names is greeted.

names = ["Anna", "Bret", "Clara"]
for name in names:
    print("Hello, " + name + "!")


Output 2:

Hello, Anna!
Hello, Bret!
Hello, Clara!


Example 3:
A for loop can also go through each position in a text. In this code, each character of the word "Banana" is printed individually.

for x in "Banana":
  print(x)

The range() Function

To repeat a code a specific number of times, you can use the range() function.


The range() function generates a sequence of numbers that by default starts at 0, increases in steps of 1, and stops before the specified end number. However, you can specify a different start value, step size, or end number.


Example 1: Example 2: Example 3: 
for x in range(5):
  print(x)
 
for x in range(2, 7):
  print(x)
 
for x in range(2, 30, 6):
  print(x)

Example 4:
You can also use the range() function to execute an action multiple times without needing a specific item in the sequence. For this, you can simply use an underscore _ as a placeholder:
