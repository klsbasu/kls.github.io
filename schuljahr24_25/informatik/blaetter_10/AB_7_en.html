Functions in Python

A function is a section of code that performs a specific task and can be reused. Functions help to structure code better and make it more readable by clearly separating specific tasks.

Defining and Calling Functions

A function is defined with the keyword def, followed by a name and parentheses (). The function’s code block is then indented.


Example:
The function greeting is defined and then called.

def greeting():
    print("Hello! Welcome to Python.")

greeting()


Output:

Hello! Welcome to Python.

Functions with a Parameter

With parameters, we can pass values to a function, which it can then use for its task.


Example:
The function greet_person takes a name as a parameter. When we call greet_person("Jojo"), the function uses "Jojo" in the greeting text and outputs "Hello, Jojo!".

def greet_person(name):
    print("Hello "  + name + "!")

greet_person("Jojo")


Output:

Hello Jojo!

Functions with Multiple Parameters

Functions can also have multiple parameters to allow for more complex calculations or tasks.


Example 1:
This function add_numbers takes two numbers as parameters and outputs their sum.

def add_numbers(a, b):
    print("The sum is", a + b)

add_numbers(3, 4)


Output 1:

The sum is 7


Example 2:
This function takes a person's name and age and outputs a personalized greeting.

def greet_with_age(name, age):
    print("Hello", name + ", you are", age, "years old.")

greet_with_age("Lisa", 30)


Output 2:

Hello Lisa, you are 30 years old.

Functions with a Return Value

A function can also have a return value that returns the result of a calculation to the main program. For this, the keyword return is used.


Example:
The function double doubles a number and returns the result.

def double(x):
    return x * 2

result = double(5)
print(result)


Output:

10


Note: It is not enough to simply call the function with double(5). The function returns a value. To output it, you need to use the print() command. If you want the function to directly output the doubled number, you must define it without return:

def double(x):
    print(x * 2)


This section, however, focuses on defining functions with return values.
