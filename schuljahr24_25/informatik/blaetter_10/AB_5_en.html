Task 1
Calculating Discounts

Write a program that asks the user for the purchase price and based on this price calculates the corresponding discount. The following conditions apply:

  • Price > 100: 10% discount.
  • 50 < Price <= 100: 5% discount.
  • Price <= 50: no discount.

Calculate the final price after subtracting the discount and display it.

Tip: If the purchase price is 80 Euros and the discount is 5%, the formula is: Final price = 80 × (1 − 0.05) = 80 × 0.95 = 76. The final price after subtracting the 5% discount is 76 Euros.


Task 2
Checking a Number
Write a program that reads a number from the user and checks:
  • Whether the number is even.
  • Whether the number is divisible by 5.

In the end, the program should output a corresponding message for each condition.

Tip: To check whether a number is divisible by another number, you can use the modulo operator %. The modulo operator returns the remainder of a division. If the remainder is 0, then the number is divisible by the other number without a remainder. For example, if number % 2 == 0 holds true, number is even.

Task 4
Convert Temperatures
Write a program that reads a temperature in Fahrenheit from the user and converts it to Celsius. If the converted temperature is below 0°C, the program should display a warning.
Use the following formula to convert: Celsius = (Fahrenheit - 32) * 5/9.


Task 5
Body Mass Index (BMI)
Write a program that calculates a person's Body Mass Index (BMI).
The user's first name, last name, weight (in kg), and height (in m) should be entered.

Use the following formula to calculate BMI:
BMI = Weight / (Height * Height)

The program should calculate the BMI and output an evaluation based on the following table:
BMI Value Evaluation
under 19 Underweight
up to 25 Normal weight
up to 29 Slight overweight
up to 39 Overweight
over 39 Extreme overweight


Example output:

Please enter your details!

First name: Max
Last name: Meier
Weight (in kg): 75
Height (in m): 1.85
===============================
Max Meier
---------
BMI: 21.91
Evaluation: Normal weight


Task 6
Population Growth Calculation

Write a program that calculates the development of a mouse population after one step.
The population is divided into three age groups: young, adult, and old mice.


Diagram for Population Growth:
The following diagram shows how the mouse population develops in different age groups: 
    flowchart TD
        A[Young mice] -->|1/2| B[Adult mice]
        B -->|1/3| C[Old mice]
        B -->|4| A
        C -->|2| A


More specifically:
New young mice = 4 * Number of adults + 2 * Number of old ones
New adult mice = Number of young // 2 (Integer division)
New old mice = Number of adults // 3


Your program should work as follows:

Enter the initial population values!
Number of young mice: 6
Number of adult mice: 9
Number of old mice: 12

The calculated population values are:
Number of young mice: 60
Number of adult mice: 3
Number of old mice: 3
