Kommentare

In Python sind Kommentare Textteile im Code, die nicht ausgeführt werden. Sie helfen, den Code zu erklären oder zu dokumentieren.

Einzeilige Kommentare

Sie beginnen mit einem #, und alles danach wird ignoriert.

# Dies ist ein Kommentar, der nicht ausgeführt wird
print("Hallo Welt")  # Dies ist ein Kommentar neben einer Codezeile

Ausgabe im Terminal:

Hallo Welt

Erklärung: Die erste Zeile wird komplett ignoriert, und der Text rechts vom # wird ebenfalls ignoriert, während die print()-Funktion ausgeführt wird.

Mehrzeilige Kommentare:

Sie beginnen und enden mit drei Anführungszeichen """. Alles dazwischen wird ignoriert.

"""
Das ist ein
mehrzeiliger Kommentar.
Er erstreckt sich über mehrere Zeilen.
"""
print("Python ist super!")

Ausgabe im Terminal:

Python ist super!

Erklärung: Der gesamte Text zwischen den dreifachen Anführungszeichen wird ignoriert, und der print()-Befehl wird ausgeführt.

Aufgaben

  1. Füge einen einzeiligen Kommentar zu folgendem Code hinzu, der erklärt, was der Code macht:

Berechnungsoperatoren

Mit Berechnungsoperatoren kannst du mathematische Operationen wie Addition, Subtraktion, Multiplikation und Division durchführen.

Zusätzlich ermöglicht der Operator ** das Potenzieren von Zahlen.

Der Operator // führt eine Ganzzahldivision durch, das heißt, das Ergebnis wird als ganze Zahl zurückgegeben.

Beim Modulo-Operator % wird ebenfalls eine Division ausgeführt, jedoch gibt er nur den Rest der Division zurück.

Operation Beispiel Ausgabe
+ Addition print(10 + 3) 13
- Subtraktion print(10 - 3) 7
* Multiplikation print(10 * 3) 30
** Potenzieren print(10 ** 3) 1000
/ Division print(10 / 3) 3.33333333333
// Ganzzahldivision print(10 // 3) 3
% Modulo-Operator print(10 % 3) 1

Strings in Python

In Python sind Strings (Zeichenketten) eine Folge von Zeichen, die als Text dargestellt werden. Sie werden von doppelten Anführungszeichen " " umschlossen. Du kannst sie aber auch von einfachen Anführungszeichen ' ' umschließen. Strings können Buchstaben, Zahlen, Symbole oder eine Kombination davon enthalten.

Verkettung mit dem + Operator

Mit dem + Operator kannst du Strings miteinander verbinden. Dies nennt man auch „Verketten“.

Bemerkung: Wenn du den + Operator verwendest, werden die Strings direkt aneinandergefügt, ohne dass automatisch Leerzeichen zwischen den Werten eingefügt werden. Um ein Leerzeichen hinzuzufügen, musst du es explizit als " " in die Verkettung einfügen.

Beispiel:

print("Hallo" + "Welt")
print("Hallo" + " " + "Welt")

Ausgabe im Terminal:

HalloWelt
Hallo Welt

Was sind Variablen?

Variablen sind Speicherorte für Informationen, die später im Programm verwendet werden können.

Beispiel:

name = "Lara"
age = 40

Bemerkung: Text (wie „Lara“) muss immer in Anführungszeichen stehen, während Zahlen ohne Anführungszeichen geschrieben werden.

Du kannst auch Variablen, die du definiert hast, mit der print()-Funktion ausgeben:

name = 'Lara'
age = 40
print(name)
print(age)

Ausgabe im Terminal:

Lara
40

Wie definiert man Variablen in Python?

Eine Variable erhält einen Namen und einen Wert, die durch den Zuweisungsoperator = (Gleichheitszeichen) miteinander verbunden sind. Beim Benennen einer Variablen gibt es jedoch einige Regeln, die du beachten musst:

  • Ein Variablenname muss mit einem Buchstaben oder einem Unterstrich beginnen, aber niemals mit einer Zahl.
  • Ein Variablenname darf keine Leerzeichen oder Sonderzeichen wie Umlaute (ä, ö, ü) enthalten.
  • Du darfst keine reservierten Schlüsselwörter wie class, if oder while als Variablennamen verwenden.
  • Python unterscheidet zwischen Groß- und Kleinschreibung. Das bedeutet, dass name und Name zwei verschiedene Variablen sind.

Überschreiben von Variablenwerten

In Python kannst du den Wert einer Variable jederzeit ändern. Das nennt man Überschreiben. Wenn du einer bereits existierenden Variablen einen neuen Wert zuweist, wird der alte Wert durch den neuen ersetzt.

x = 10  
print(x)  

x = 20  
print(x)

Ausgabe im Terminal:

10
20

Erklärung: Zuerst hat die Variable x den Wert 10. Danach wird sie mit dem Wert 20 überschrieben. Der alte Wert (10) wird also komplett ersetzt.

Bemerkung: - Du kannst Variablen so oft überschreiben, wie du möchtest. - Der neue Wert muss nicht den gleichen Datentyp wie der alte Wert haben. Du kannst zum Beispiel einen String in eine Variable speichern, die vorher eine Zahl war.

x = 100  
print(x)

x = "Hallo"  
print(x)

Datentypen

In Python gibt es verschiedene Datentypen, die bestimmen, wie Daten behandelt werden. Wir haben bereits drei kennengelernt.

String (Zeichenkette): Ein Text, der in Anführungszeichen steht.

text = "Hallo, Welt!"

Integer (Ganzzahl): Eine Zahl ohne Dezimalstellen.

zahl = 42

Float (Gleitkommazahl): Eine Zahl mit Dezimalstellen.

pi = 3.14

Mit der Funktion type() kannst du herausfinden, welcher Datentyp ein Wert hat:

print(type("Hallo Welt"))
print(type(42))
print(type(3.14))

<class 'str'>
<class 'int'>
<class 'float'>

Eingabe von Werten: Die Funktion input() in Python

Manchmal möchtest du, dass der Benutzer deines Programms dir Informationen gibt, wie z.B. seinen Namen, sein Alter oder seinen Wohnort. Dafür kannst du die Funktion input() verwenden:

wohnort = input("Wo wohnst du? ")
print(wohnort)

Wenn das Programm ausgeführt wird, erscheint Folgendes im Terminal:

Wo wohnst du?

Der Benutzer kann dann eine Eingabe machen. Sobald der Benutzer die Enter-Taste drückt, wird die Eingabe in der Variable wohnort gespeichert und anschließend mit print() ausgegeben.

Beispielablauf:

Wo wohnst du? Berlin
Berlin

Erklärung: Das Programm zeigt dem Benutzer die Frage „Wo wohnst du?“ im Terminal an und wartet auf eine Eingabe. Alles, was der Benutzer eingibt (egal ob Text oder Zahlen), wird als String in der Variable wohnort gespeichert und danach mit der print()-Funktion im Terminal ausgegeben.

Datentypen umwandeln

In Python müssen Daten manchmal von einem Typ in einen anderen umgewandelt werden. Zum Beispiel möchtest du vielleicht eine Zahl in einen Text (String) umwandeln oder einen Text, der eine Zahl darstellt, in eine Ganzzahl konvertieren. Dafür stehen in Python spezielle Funktionen zur Verfügung:

str() wandelt eine Zahl in einen Text (String) um.

zahl = 42
text = str(zahl)

int() wandelt einen Text, der nur aus einer Zahl besteht, in eine Ganzzahl um. Achtung: Der Text muss vollständig eine Zahl sein, sonst tritt ein Fehler auf.

text = "123"
zahl = int(text)

float() wandelt einen Text oder eine Ganzzahl in eine Gleitkommazahl (float) um. Dies ist nützlich, wenn du mit Dezimalzahlen arbeiten möchtest.

text = "3.14"
zahl = float(text)
