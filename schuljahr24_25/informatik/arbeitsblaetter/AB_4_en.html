Nested if Statements

You can have if statements inside other if statements. This is called nested if statements.


Example:

x = 41
if x > 10:
    print("Over ten,")
    if x > 20:
        print("and also over 20!")
    else:
        print("but not over 20.")


Explanation: The code first checks if x is greater than 10. If so, it prints "Over ten,". Then it checks if x is greater than 20. If true, it prints: "and also over 20!"; otherwise: "but not over 20."


Since x = 41, both conditions are true, so the following is printed:

Over ten,
and also over 20!


Always pay attention to indentation!
